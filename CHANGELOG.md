# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-31

### Added

- Initial release
- HTTP server on port 9876
- DOM commands: navigate, click, type, clear, getText, getValue, getAttribute, exists, waitFor, select, check, uncheck, focus, blur, pressKey, scrollTo, submit, wait, eval, getElements, getHtml, getUrl, getTitle
- Screenshot capture via html2canvas
- JavaScript/TypeScript API package
