# tauri-plugin-automation-api

JavaScript/TypeScript API for [tauri-plugin-automation](https://github.com/dcherrera/tauri-plugin-automation).

## Installation

```bash
npm install tauri-plugin-automation-api
```

## Usage

```typescript
import { initAutomation } from 'tauri-plugin-automation-api'

// After your app mounts
initAutomation()
```

## Requirements

- `@tauri-apps/api` ^1.5.0
- Rust plugin must be installed (see main repo)
